# Weather API Visualization Project

This project fetches real-time weather data from the OpenWeatherMap API
and visualizes it using Python.

# Technologies Used
- Python
- Requests
- Pandas
- Matplotlib
- Seaborn

# Features
- Fetches weather data for multiple cities
- Displays temperature, humidity, and wind speed
- Generates a visualization dashboard

# How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your OpenWeatherMap API key in main.py
3. Run the script:
   python main.py

# Output
The weather visualization dashboard is saved in the images folder.
